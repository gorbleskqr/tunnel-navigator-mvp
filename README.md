# Tunnel Navigator MVP

Mobile-first campus tunnel routing prototype built with Expo/React Native.

## Quick Start

1. Install dependencies: `npm install`
2. Start dev server: `npm run start`
3. Run on device: `npm run android` or `npm run ios`
4. Run web target: `npm run web`
